Требования:

Версия опенкарта: 2.x.x.х
для использования модуля вы должны быть клиентом компании shop-logistics


Установка:

1) Скопируйте файлы модуля из папки /upload на Ваш сервер
   или с помощью "Установки расширений" 

3) Установите следующие модули
Модули / Расширения
Выбрать "Доствка"
Активируйте модуль - ShopLogistics - курьерская доставка

3) Настройте модуль доставки и установите API ID (его нужно взять в личном кабинете Shop-logistics в разделе "Ваши данные")
